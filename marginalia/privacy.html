旁注 (Marginalia)

Privacy Policy / 隐私政策

Last updated: June 2026

中文

旁注（PangZhu）是一个 Microsoft Edge 侧边栏扩展,帮助你用自己电脑上已安装的 Claude Code 阅读当前网页。

我们收集什么

开发者不收集、不存储、不接收你的任何数据。开发者不运营任何服务器。

数据如何处理

当你主动点击某个动作（总结 / 翻译 / 提问等)时,旁注读取你当前网页的文字、你选中的内容,以及（仅当你开启"看图"时)当前屏幕截图,并通过本机的原生消息(Native Messaging)通道,仅发送给你自己电脑上已安装并登录的 Claude Code 命令行进行处理。这些内容不会发送到开发者或任何第三方服务器。

本地存储

你的设置与对话历史仅保存在你本地浏览器中,不上传。

可选的"联网搜索"

由你本机的 Claude 自行发起,遵循 Anthropic 的相应条款;旁注本身不进行任何网络请求,也无任何遥测。

联系方式

mumu@moonstack.org

English

旁注 (Marginalia / PangZhu) is a Microsoft Edge side-panel extension that helps you read the current web page using the Claude Code CLI installed on your own computer.

What we collect

The developer collects, stores, and receives no data. The developer operates no server.

How data is handled

When you actively trigger an action (summarize / translate / ask, etc.), Marginalia reads the current page's text, your selected text, and (only if you enable the screenshot option) a screenshot of the current screen, and sends them — solely to the Claude Code CLI installed on your own computer, via the local Native Messaging channel. This content is never sent to the developer or any third-party server.

Local storage

Your settings and conversation history are stored only locally in your browser and are never uploaded.

Optional web search

When enabled, web search is performed by Claude on your machine under Anthropic's terms. Marginalia itself makes no network requests and contains no telemetry.
